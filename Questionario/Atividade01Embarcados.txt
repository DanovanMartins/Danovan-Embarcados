1) O que ão sistemas embarcados?

	São sistemas completos e independentes, com desempenho próximo a um computador, com um propósito determinado pelo usuário.

2) O que são sistemas microprocessados?

	São sistemas capazes de receber, processar e enviar dados.

3) Apresente aplicações de sistemas embarcados:
	
	a) Para a indústria automotiva;
	
	Incrementar sistemas capazes de controlar funções em um automóvel, tais como abrir janelas, controlar freio de disco, direção elétrica, etc.

	b) Para eletrodomésticos;
	
	Controlar dispositivos eletrodomésticos através de um dispositivo, ativar funções do aparelho de acordo com um horário, etc.

	c) Para automação industrial.

	Automatizar e otimizar o processo industrial, por exemplo, fazer com que um equipamento fique responsável por realizar um processo de forma precisa, mesmo que seja apenas um processo.

4) Cite arquiteturas possíveis e as diferenças entre elas.
	
	AVR - são microcontroladores de 8 bits, desenvolvidos sob a tecnologia RISC - Reduced Instruction Set Computer (Computador com Set de Instruções Reduzido) e arquitetura HAWARD que separa a memória de dados da memória de programa.

	ARM – É um processador baseado na arquitetura RISC (Reduced Instruction Set Computer) de baixa energia capaz de processar dados através de instruções simples. 

5) Por que usamos o Raspberry Pi na disciplina, ao invés de outro system on chip?

	O Raspberry Pi possui alto poder de processamento sendo capaz de utilizar um sistema operacional baseado no linux, pode ser conectado a um monitor juntamente com periféricos de saída como mouse e teclado. Desta forma o Raspberry pode ser considerado a um micro computador. Além disto, poremos utilizar as GPIO para comunicar com dispositivos eletrônicos, fazendo com que seja possível criar projetos embarcados. O raspberry é utilizado na disciplina porque possui vários dos requisitos necessáros para criar um sistema embarcado,isto através de um conjunto de componentes contidos em uma pequena placa.
